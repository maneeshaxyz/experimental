# Code of Conduct

## Our standard

Please be respectful, constructive, and professional in all project interactions.

## Expected behavior

- Use welcoming and inclusive language.
- Focus feedback on ideas, code, and behavior.
- Assume good intent and ask clarifying questions before escalating.

## Unacceptable behavior

- Harassment, insults, or personal attacks.
- Discrimination or exclusionary language.
- Repeated off-topic disruption or bad-faith engagement.

## Reporting

If you run into a conduct issue, open a private report with the maintainers through the repository contact method you use for sensitive concerns.
