# Contributing

Thanks for contributing to this repository.

## How to contribute

1. Open an issue if you want to report a bug or discuss a change.
2. Keep issue comments focused on the specific problem or proposal.
3. When sharing code, include the smallest reproducible example you can.
4. Be clear about expected behavior, actual behavior, and any relevant context.

## Pull requests

1. Keep changes narrowly scoped.
2. Add or update tests when behavior changes.
3. Describe the motivation and the impact of the change in the pull request.

## Communication

Be respectful, assume good intent, and follow the [Code of Conduct](CODE_OF_CONDUCT.md).
